update_config=1

network={
    ssid="yourssid"
    psk="yourpskpassword"
    key_mgmt=WPA-PSK
}

### Ethernet

Connect raspberry pi to your local network

Run the ping command:
```
ping raspberrypi.local
```

Connect via ssh:
```
ssh pi@raspberrypi.local
```
